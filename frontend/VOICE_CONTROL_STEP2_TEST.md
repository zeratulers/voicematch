## 🎯 第二阶段测试：指令内容识别

### 🧪 测试步骤

1. **启动语音控制**
   - 选择患者并锁定
   - 启用语音控制
   - 点击"开始录音"

2. **唤醒系统**
   - 说："发出指令"
   - 确认：听到ding声，界面变橙色

3. **等待重启**
   - 看到"已唤醒，请说出指令..."
   - 等待1.5秒让系统重启语音识别
   - 观察实时转录区域显示"请说出指令内容..."

4. **说出指令**
   - 说出您可用指令中的内容，例如：
     - "慢慢躺在床上"
     - "往上躺一点"  
     - "放松不要紧张"
     - "眼睛看上面"

### 🔧 如果指令还是无法识别

**检查以下几点：**

1. **时机问题**：
   - 确保在说指令前等待1.5秒
   - 等到实时转录显示"请说出指令内容..."

2. **发音问题**：
   - 使用标准普通话
   - 说话清晰、语速适中
   - 尝试说完整的指令名称

3. **指令匹配**：
   - 确保说的内容与可用指令列表中的文本相近
   - 可以先尝试手动点击"播放"按钮测试音频

4. **查看日志**：
   - 重点关注是否出现"🔍 检查指令内容条件"
   - 检查 `shouldCheck` 是否为 `true`
   - 查看最终指令文本是否正确提取
